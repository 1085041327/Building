## 📐 计算公式说明

### 词条计算公式

```
最终百分比 = [固定百分比 + 获取量 / (获取量 + 固定除数)] × (1 + 总体提升百分比)

其中：
获取量 = 固定值总和 × (1 + 值提升百分比)

固定除数：
- 暴击、急速、幸运、精通：50000
- 全能：28000（特殊）
```

### 公式解析

1. **固定值总和**：所有装备上该词条的数值相加
2. **值提升百分比**：影响获取量的计算（例如：暴击值提升10%）
3. **获取量**：经过值提升后的实际数值
4. **固定百分比**：该词条的基础百分比（例如：暴击固定5%）
5. **总体提升百分比**：最终结果的额外加成（例如：暴击总体提升15%）

### 计算示例

#### 示例1：暴击属性
- 固定值总和：1000
- 暴击值提升：10%
- 暴击固定百分比：5%
- 暴击总体提升：15%
- 固定除数：50000

计算过程：
```
1. 获取量 = 1000 × (1 + 10%) = 1100
2. 转换百分比 = 1100 / (1100 + 50000) × 100 = 2.15%
3. 最终百分比 = (5% + 2.15%) × (1 + 15%) = 8.22%
```

#### 示例2：全能属性
- 固定值总和：1000
- 全能值提升：10%
- 全能固定百分比：0%
- 全能总体提升：15%
- 固定除数：28000（特殊）

计算过程：
```
1. 获取量 = 1000 × (1 + 10%) = 1100
2. 转换百分比 = 1100 / (1100 + 28000) × 100 = 3.79%
3. 最终百分比 = (0% + 3.79%) × (1 + 15%) = 4.35%
```

**注意**：全能使用更小的固定除数(28000)，因此相同数值下收益更高。

## 🚀 快速开始

### 使用方法
1. 直接用浏览器打开 `index.html` 文件
2. 无需安装任何依赖
3. 数据自动保存在浏览器本地


### 基本操作

#### 1. 配置词条百分比
在词条百分比配置区域，每个词条有4个输入框：
- **固定值获取**：来自天赋、BUFF等的固定数值（新增功能）
- **固定%**：该词条的基础百分比（默认暴击为5%，其他为0%）
- **值提升%**：影响获取量的计算
- **总体提升%**：影响最终计算结果

#### 2. 固定值获取的作用
固定值获取会直接加到该词条的固定值总和中：
```
固定值总和 = 装备提供的数值 + 固定值获取
```
适用场景：
- 天赋/被动技能提供的固定数值
- BUFF效果的固定加成
- 角色基础属性
- 特殊道具加成

#### 3. 填写装备属性
对于每件装备：
- 勾选"启用"激活该装备槽
- 选择属性类型（暴击/急速/幸运/精通/全能）
- 填写属性数值
- 武器额外有两个"额外"属性槽

#### 4. 查看统计结果
底部统计面板实时显示：
- 固定值总和
- 获取量
- 最终百分比

#### 5. 保存方案
- 点击"保存方案"按钮
- 输入方案名称
- 保存成功后可随时加载

## 💡 使用技巧

### 1. 全能词条的特殊优势 ⭐
- **高性价比**：全能使用更小的固定除数(28000)，收益比其他词条高40%-74%
- **前期推荐**：在装备数值较低时，优先堆全能性价比最高
- **全程优势**：无论数值高低，全能的转换百分比都优于其他词条

### 2. 合理配置提升百分比
- **值提升**：影响前期获取量，对低数值影响较大
- **总体提升**：影响最终结果，对所有情况都有效果
- **配合使用**：两种提升百分比搭配使用效果最佳

### 3. 注意固定百分比设置
- 根据游戏实际情况设置每个词条的固定百分比
- 暴击默认5%是示例值，请根据实际调整

### 4. 词条选择策略
- **前期**（数值<15000）：优先堆全能，性价比最高
- **中期**（数值15000-35000）：全能+暴击/急速组合
- **后期**（数值>35000）：均衡发展，全能仍有优势

### 5. 利用方案管理
- 保存多套配装方案进行对比
- 方案名称建议包含流派信息（如"暴击流"、"全能流"等）

### 6. 定期导出备份
- 使用导出功能定期备份重要配置
- 防止浏览器缓存清理导致数据丢失

## 📖 常见问题

### Q: 计算结果和游戏内不一致？
A: 请检查以下设置：
1. 固定百分比是否正确
2. 提升百分比是否遗漏
3. 装备数值是否填写正确
4. 是否有装备忘记启用

### Q: 数据会丢失吗？
A: 系统使用浏览器本地存储，除非清除缓存，否则数据不会丢失。建议定期导出备份。

### Q: 可以在手机上使用吗？
A: 可以！本工具完全响应式设计，在手机上也能完美使用。

### Q: 如何分享配装给朋友？
A: 使用"导出数据"功能导出JSON文件，发送给朋友后，对方使用"导入数据"功能即可。
